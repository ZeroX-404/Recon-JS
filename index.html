CLI — Referensi Command
Command-command yang dipakai sebelum paste ke tool ini. Workflow lengkap dari awal recon sampai ambil semua JS.
Step 1 — Temukan Semua JS Files
terminal 
# getJS — extract semua URL .js dari target
getJS --url https://target.com --output js-urls.txt

# Katana — crawl lebih dalam dengan JS parsing
katana -u https://target.com -jc -o endpoints.txt

# gospider — spider dengan js filter
gospider -s https://target.com -t 10 --js | grep "\.js"
Step 2 — Download Isi JS
terminal 
# Download semua JS dari daftar URL
cat js-urls.txt | xargs -I{} curl -s {} -o {}.js

# Atau gabung semua jadi satu file
while read url; do curl -s "$url" >> all.js; done < js-urls.txt

# Lalu paste isi all.js ke Step 1 (JS Analyzer)
Step 3 — Secret Scanning CLI (alternatif)
terminal 
# Trufflehog — scan secrets dengan verified mode
trufflehog filesystem ./out/ --only-verified

# Gitleaks — scan directory
gitleaks detect --source ./out/ --report-format json

# Grep manual untuk Cognito
grep -rE "userPoolId|identityPoolId|clientId" ./out/

# Grep manual untuk AWS keys
grep -rE "(AKIA|ASIA)[A-Z0-9]{16}" ./out/
Step 4 — Source Map Extraction
terminal 
# Extract source map → source code asli
sourcemapper -url https://target.com/main.abc.js -output out/

# Test apakah .map ada (HTTP 200 = jackpot!)
curl -I https://target.com/app.abc.js.map

# Fuzz source map candidates dari list (export dari Step 3)
ffuf -u https://target.com/FUZZ -w sourcemap-candidates.txt -mc 200
Tips Penting
